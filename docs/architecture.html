Wolfee-Watcher — схема взаимодействия компонентов

Платформа безопасности Kubernetes (стиль StackRox). Перетаскивайте компоненты мышью, чтобы менять схему. Кликните на компонент, чтобы подсветить его связи. Кликайте по легенде, чтобы скрыть/показать типы связей.

Типы связей

Наведите или кликните на компонент, чтобы увидеть описание.
Зоны сами подстраиваются под компоненты при перетаскивании. Клик по компоненту закрепляет подсветку и показывает якоря на его стрелках: тяните якорь (он прилипает к сетке), двойной клик по линии добавляет ещё один якорь, двойной клик по якорю удаляет его — так стрелку можно вести ломаной в обход блоков. Раскладка (позиции, изгибы, тема) сохраняется автоматически; «Сбросить расположение» возвращает всё к исходному.

Всё разворачивается через Helm в namespace wolfee-watcher. Между Go-сервисами — обязательный mTLS (pkg/mtls), сертификаты выдаёт cert-server (связи скрыты по умолчанию — включите в легенде).
